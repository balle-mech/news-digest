### [★★★★☆] Claude Code GitHub Action脆弱性 — 1件のIssueでリポジトリ乗っ取りが可能（CVSSv4 7.8・修正済み）
**ソース:** [The Hacker News](https://thehackernews.com/2026/06/claude-code-github-action-flaw-let-one.html) / [GMO Flatt Security](https://flatt.tech/research/posts/poisoning-claude-code-one-github-issue-to-break-the-supply-chain/)
